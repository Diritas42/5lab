# 5lab
# Описание модулей
# settings.py
Хранит все глобальные настройки и конфигурации игры.
Содержит: размеры экрана и игрового поля: SCREEN_WIDTH, SCREEN_HEIGHT - размеры основного окна, GAME_WIDTH, GAME_HEIGHT - размеры игрового поля, GAME_OFFSET_X, GAME_OFFSET_Y - отступы игрового поля от краев окна. Цветовая палитра: основные цвета игры (BLACK, GREEN, RED, BLUE, etc.), цвета для различных состояний (ALERT_RED, ALERT_YELLOW), полупрозрачные цвета для UI элементов. Параметры объектов: размеры игрока, врагов, документов, выхода, скорости движения для разных режимов, параметры обнаружения (дальность, угол обзора).
# main.py
Точка входа приложения, управляет главным игровым циклом.
Содержит: инициализацию Pygame, создание экземпляра главного класса Game, главный игровой цикл с контролем FPS, обработку завершения работы приложения.
Ключевые функции: main() - функция запуска игры, бесконечный цикл, пока игра работает, вызов методов обновления и отрисовки игры.
# game.py
Основной управляющий класс всей игры.
Содержит: инициализацию игровых компонентов: создание окна Pygame, инициализация шрифтов для UI, загрузка уровней. Управление состоянием игры: game_state (playing, levelComplete, gameOver), уровень обнаружения игрока, режимы тревоги и усиленной бдительности. Игровую логику: обработка ввода пользователя, обновление всех игровых объектов, проверка условий победы/поражения, управление переходами между уровнями. Систему рендеринга: отрисовка игрового поля на отдельной поверхности, рендеринг UI элементов, отладочная информация.
# player.py
Класс, представляющий игрового персонажа.
Содержит: свойства игрока: позиция, размер, скорость, направление движения (0-вправо, 1-влево, 2-вверх, 3-вниз), счетчик столкновений для отладки. Механику движения: обработка ввода с клавиатуры (WASD + Shift), система коллизий со стенами, ограничение движения в пределах игрового поля. Визуальное представление: отрисовка тела игрока, индикатор направления (голова).
# enemy.py
Класс искусственного интеллекта врагов.
Содержит: систему поведения: патрулирование по заданным маршрутам, преследование игрока при обнаружении, коллизии с другими врагами и стенами. Механику обнаружения: конус видимости с ограниченным углом (60 градусов), проверка прямой видимости (line of sight), усиленная бдительность при массовом устранении. Визуальные элементы: отрисовка конуса видимости, индикаторы состояния (тревога, усиленная бдительность), глаза, показывающие направление взгляда.
# level.py
Описание и отрисовка игровых уровней
Содержит: структуру уровня: массив стен (координаты и размеры), данные о врагах (позиции, маршруты патрулирования), позиция выхода и стартовая позиция игрока, расположение документов. Методы рендеринга: отрисовка стен, отображение выхода с текстовой меткой.
# document.py
Класс для игровых объектов - секретных документов.
Содержит: свойства документа: позиция, размер, цвет, состояние (собран/не собран). Логику взаимодействия: проверка коллизии с игроком, механика сбора документа. Визуальное представление: золотой прямоугольник с текстом "DOC", эффект блеска для привлекательности.
